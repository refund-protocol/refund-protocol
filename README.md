REFUND is recovery infrastructure designed to address systemic losses caused by rug pulls and malicious token behavior on Solana.

Losses created by rugs are typically treated as final. REFUND exists to challenge that assumption by treating rugs as recoverable events rather than terminal failures.

REFUND is not charity.  
REFUND is not discretionary.  
REFUND does not rely on offchain trust.

The protocol operates through autonomous classification, adversarial extraction, and deterministic recovery flows enforced by onchain constraints.
